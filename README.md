# tsno

[![npm version](https://badgen.net/npm/v/tsno)](https://npm.im/tsno)

## Install

```bash
npm i tsno -D
```

## Usage

```bash
tsno script.ts
```

Supported files: `.js` `.jsx` `.ts` `.tsx` `.cjs` `.mjs` `.json`

All `node` flags are forwarded, i.e. `tsno script.ts --experimental-wasm-modules` works as expected.
